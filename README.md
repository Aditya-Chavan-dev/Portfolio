# Portfolio Monorepo

## Overview
This is a comprehensive portfolio system separated into frontend, backend, and firebase configuration.

## Structure
- **frontend/**: React + Vite application (deployed to Firebase Hosting).
- **backend/**: Node.js + Express API (deployed to Render).
- **firebase/**: Firebase configuration and rules.
- **projects/**, **recruiterNotes/**, **analytics/**: Placeholders for future modules.

## setup
1.  **Environment Variables**: Copy `.env.example` to `.env` and fill in secrets.
2.  **Backend**: `cd backend && npm install && npm run dev`
3.  **Frontend**: `cd frontend && npm install && npm run dev`
